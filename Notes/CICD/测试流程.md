# 3. 对项目代码做一些修改

我推荐你做这样的修改：访问 `public/res/` 目录，在其中添加 `image_*.png` (\*是 1~9 中任意数字)，然后提交更改并推送到 gitlab

# 4. 查看 Jenkins 后台流水线是否正常工作

打开 Jenkins，查看项目左下角的 Builds，会发现多了一次 Build, 可以点进去看 Console Output 的相关信息，如果看到界面打绿色的勾就是 build 成功了
