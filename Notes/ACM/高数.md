# 齐次线性方程

对于二阶齐次线性差分方程  
$$a y_{n+2} + b y_{n+1} + c y_n = 0$$
假定解的形式为 $y_n = r^n$，代入原方程，得特征方程：  
$$a r^2 + b r + c = 0$$
求其特征根，分以下情形讨论：  

### ① 当判别式 $\Delta > 0$  

特征方程有两个不相等的实根 $r_1$ 与 $r_2$，差分方程的**通解**为：  
$$y_n = C_1 r_1^n + C_2 r_2^n$$

### ② 当判别式 $\Delta = 0$   

特征方程有二重实根 $r_0$，差分方程的**通解**为：  
$$y_n = (C_1 + C_2 n) r_0^n$$

### ③ 当判别式 $\Delta < 0$ 时  

特征方程有一对共轭复根 $r_1 = \alpha + \beta i$，$r_2 = \alpha - \beta i$（其中 $\alpha, \beta \in \mathbb{R}$，$i$ 为虚数单位）。  

令 $\rho = \sqrt{\alpha^2 + \beta^2}$（复根的模），$\theta = \arg(r_1) = \arctan\left( \frac{\beta}{\alpha} \right)$（复根的辐角），则差分方程的**通解**可表示为：  
$$y_n = \rho^n \left( C_1 \cos(n\theta) + C_2 \sin(n\theta) \right)$$  

或等价的三角形式：  
$$y_n = A \rho^n \cos(n\theta + \phi)$$  
（其中 $A, \phi$ 为由初始条件确定的常数）
